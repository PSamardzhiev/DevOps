[Install Docker]
	
To install docker on Debian Server, we need to execute the following commands:
	sudo apt-get update
	sudo apt-get upgrade
	sudo apt-get install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
